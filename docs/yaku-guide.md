# 役种一览 - 功能使用文档

## 1. 功能介绍

役种一览是雀魂麻将攻略系统的核心模块之一，用于展示和学习麻将中的各种役种（牌型）。该功能提供了35种以上役种的详细信息，包括牌型展示、番数计算、分类筛选等功能，帮助玩家学习和记忆麻将役种。

### 主要功能特性

- **35+ 种役种展示**：涵盖一番到双倍役满的所有标准役种
- **按番数筛选**：支持按一番、二番、三番、六番、满贯、役满、双倍役满、流局等分类筛选
- **按分类筛选**：支持无限制、门前清、副露后三种役种条件筛选
- **搜索功能**：支持按役种名称和描述关键词搜索
- **熟练度管理**：点击增加已胡次数，长按连续增加或减少
- **持久化存储**：使用 localStorage 保存筛选状态和熟练度数据
- **牌型可视化**：使用麻将牌图形直观展示每种役种的牌型
- **导航侧边栏**：右侧提供快速导航列表

## 2. 使用说明

### 2.1 访问入口

在应用导航栏中找到「役种一览」入口，点击即可进入功能页面。

### 2.2 筛选役种

页面顶部提供两组筛选条件：

#### 按番数筛选

点击番数标签页进行筛选：

| 标签 | 说明 |
|------|------|
| 全部牌型 | 显示所有役种 |
| 一番 | 显示番数为1的役种 |
| 二番 | 显示番数为2的役种 |
| 三番 | 显示番数为3的役种 |
| 六番 | 显示番数为6的役种（清一色） |
| 满贯 | 显示番数为5的役种（流局满贯） |
| 役满 | 显示番数为8的标准役满 |
| 双倍役满 | 显示双倍役满役种 |
| 流局 | 显示流局相关的役种 |

#### 按分类筛选

根据役种的适用条件进行筛选：

| 分类 | 说明 |
|------|------|
| 不过滤条件 | 显示所有符合条件的役种 |
| 无限制 | 可在任何状态下达成的役种 |
| 门前清 | 必须在门前清状态下才能达成的役种 |
| 副露后 | 副露后适用的役种（副露减1番） |
| 不是役 | 仅作为加番效果的牌型（如宝牌） |

### 2.3 搜索役种

在页面右上角的搜索框中输入关键词，可以搜索役种名称或描述内容。点击「清除」按钮可清空搜索内容。

### 2.4 记录熟练度

#### 点击增加

- 点击役种卡片中的「已胡：X」区域，可增加该役种的已胡次数
- 增加时会有 +1 的动画效果

#### 长按连续操作

- **增加熟练度**：在役种卡片区域（除「已胡」区域外）长按约800毫秒，会进入连续增加模式
- **减少熟练度**：在「已胡：X」区域长按约800毫秒，会进入连续减少模式
- 长按操作会有进度条动画提示

> 注意：仅标记为 `isHu: true` 的役种才支持熟练度管理功能。

### 2.5 导航侧边栏

页面右侧提供导航侧边栏，显示当前筛选结果中的所有役种名称列表。点击列表项可快速跳转到对应的役种卡片位置。

## 3. 核心概念

### 3.1 番数系统

麻将中的番数是衡量役种价值的重要指标。不同番数对应不同的得分倍数：

| 番数 | 类别 | 代表役种 |
|------|------|----------|
| 1-3 番 | 基础役种 | 立直、断幺九、平和、一杯口等 |
| 5 番 | 满贯 | 流局满贯 |
| 6 番 | 跳满 | 清一色 |
| 8 番 | 役满 | 天和、地和、大三元、四暗刻、国士无双等 |
| -2 | 双倍役满 | 四暗刻单骑、国士无双十三面、纯正九莲宝灯、大四喜（双倍） |
| -3 | 流局 | 四风连打、四杠散了、九种九牌、四家立直 |

### 3.2 役种分类

根据达成条件，役种可分为三类：

#### 无限制

可在任何状态下达成的役种，无论是否副露（吃、碰、杠）都可以使用。例如：

- 断幺九
- 役牌（自风、场风、三元牌）
- 对对和
- 三暗刻
- 大三元

#### 门前清

必须在门前清状态下（未进行任何副露）才能达成的役种。例如：

- 立直
- 门前清自摸和
- 平和
- 一杯口
- 七对子
- 国士无双

#### 副露后

副露后可以达成，但需要注意的是副露后会减1番（仅适用于1-2番役种）。例如：

- 混全带幺九
- 一气通贯
- 三色同顺
- 纯全带幺九
- 混一色
- 清一色

### 3.3 特殊标记

部分役种具有特殊标记：

| 标记 | 说明 |
|------|------|
| 门前清限定 | 只能在门前清状态下达成 |
| 副露减1番 | 副露后达成时番数减1 |
| 庄家限定 | 只能是庄家达成（如天和） |
| 子家限定 | 只能是闲家达成（如地和） |
| 不是役 | 仅作为加番效果，不是独立役种（如宝牌） |

## 4. 调试方法

### 4.1 浏览器控制台 API

项目在全局暴露了 `yakuUtils` 对象，可在浏览器控制台（Developer Tools → Console）中直接调用以下方法：

#### 设置役种熟练度

```javascript
// 设置某个牌型已胡次数
// 参数：id 为役种唯一标识，count 为要设置的次数
yakuUtils.setYakuMastery('reach', 5)    // 立直已胡5次
yakuUtils.setYakuMastery('tanyao', 3)  // 断幺九已胡3次
yakuUtils.setYakuMastery('pinfu', 10) // 平和已胡10次
```

#### 清理所有熟练度

```javascript
// 清理所有已胡次数缓存
yakuUtils.clearAllMastery()
```

## 5. 常见问题

### Q1: 为什么有些役种没有「已胡」按钮？

只有标记为胡牌役种（`isHu: true`）才显示「已胡」计数功能。一些纯效果役种（如宝牌、赤宝牌、北宝牌）不显示此功能。

### Q2: 长按增加/减少熟练度的速度是多少？

长按触发连续操作的时间阈值为800毫秒。触发后，熟练度会持续快速增加或减少。

### Q3: 筛选状态会保存吗？

会的。页面会自动保存当前选择的番数筛选条件和分类筛选条件到 localStorage，刷新页面后会自动恢复上次的筛选状态。

### Q4: 熟练度数据会丢失吗？

熟练度数据存储在浏览器的 localStorage 中。除非清除浏览器缓存或使用隐私/无痕模式，否则数据会持久保存。

### Q5: 如何重置所有熟练度？

可以在浏览器控制台执行：

```javascript
yakuUtils.clearAllMastery()
```

或者在页面上逐个将每个役种的熟练度减至0。

### Q6: 为什么役种的分类筛选选项会变化？

分类筛选选项会根据当前选择的番数动态调整。例如选择「役满」时，会显示「无限制」和「门前清」选项（因为役满役种不涉及副露减番）；而选择「一番」时，还会额外显示「不是役」选项（因为宝牌等加番牌属于一番）。

### Q7: 导航侧边栏显示的数量和实际卡片数量不一致？

导航侧边栏显示的是当前筛选条件下的役种数量，与卡片列表数量保持一致。如果进行了搜索或筛选，导航列表会相应更新。
